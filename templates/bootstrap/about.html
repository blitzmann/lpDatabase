About

lpStore

lpStore is a web application useful in calculating the value of LP Store offers. It is written by Sable Blitzmann in PHP with a MySQL backend and using EMDR as a price data feed. The code is open-source at GitHub

History

I was first introduced to Factional Warfare when my corp joined it after a particularly horrible null sec campaign. I immediately saw the value in automatically calculating ISK/LP and went to work on lpStore's prototype. It was a simple application, with a script set to periodically run and update prices from EVE Central. I used DaOpa's Ellatha and manually copy/pasted the LP store offers for my faction - I didn't bother with other stores. Development eventually halted as my corp disbanded and absorbed into our old alliance's main corp and we moved back to null and away from FW.

The null sec thing eventually faded away again, and many of us jumped right back into FW. I was interested in continuing where I left off; however I wanted to go bigger and on a much larger scale. I no longer wanted to provide data for my FW faction and for my friends only, but for all LP Store's in-game and the rest of the EVE community.

Release

lpStore was introduced to the communitiy on April 29th, 2013, and it's source code can be viewed on GitHub.

lpDatabase

lpDatabase is a completely separate project that lpStore is built upon. lpDatabase is simply the SQL database used for lpStore, and it is currently being maintained separately due to the scope of the project.

History

Unfortunately, CCP does not release LP Store data along with their static database dumps. And so work set out collecting the data that I needed to upstart this project. I tried to search for this data online; unfortunately, it would seem that all the LP Store Database sites are outdated and proprietary (do not offer their SQL data to third parties). I finally came across usable data; thanks to Zanto Snix and his spreadsheet, I was able to download a CSV file of the database. His work looked like a scrape from DaOpa's Ellatha LP Store, with more than enough errors, but it seemed to be more or less complete and it was definitely a start. From there, I went through a long and boring process of detecting and correcting most of the errors and typos (more details on how I achieved this can be found in this reddit comment) before finally converting it to usable SQL data.

Release

lpDatabase can be found on the EVE Online Technology Lab forum. As LP Store data is based on player submission to keep it up-to-date, the project will never be marked as complete. If you ever see a discrepancy with offer information, please contact me or post in the LPDB thread

emdr-py

emdr-py is an open-source EMDR consumer script written in Python and using a Redis key-value storage backend. Initially based off of Steve Ronuken's Perl script, it's been ported to Python 2.7. It is a set-it-and-forget-it consumer, constantly pulling data from the EMDR services for the most up-to-date prices available.

Release

emdr-py is released on GitHub.
